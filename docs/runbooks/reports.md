# Reports Catalog

## Monthly Summary (JSON)
```
GET /reports/monthly-summary?year=2024&month=1
```

## Quarterly Summary (JSON, Pro)
```
GET /reports/quarterly-summary?year=2024&quarter=1
```

## Profit & Loss (JSON, Pro)
```
GET /reports/profit-loss?start_date=2024-01-01&end_date=2024-03-31
```

## Tax Year Summary (JSON, Pro)
```
GET /reports/tax-year-summary?tax_year=2023-04-06/2024-04-05
```

## Mortgage Readiness (PDF, Pro)
```
GET /reports/mortgage-readiness
```

## Reporting Cadence (JSON)
```
GET /reports/reporting-cadence
```
Returns whether quarterly reporting is recommended based on turnover.
