# ПАСПОРТ ПРОЕКТА: FinTech Platform (Эталонная архитектура)
**Статус:** ЗАВЕРШЁН
**Дата фиксации:** 2025-10-04

## 1. Видение и реализованная концепция

Проект представляет собой **полнофункциональный, готовый к промышленной эксплуатации прототип** облачной FinTech-платформы для самозанятых. Система реализует полный жизненный цикл данных: от сбора и обогащения до анализа, генерации ценных инсайтов и безопасного хранения.

Архитектура построена на современных, проверенных и масштабируемых технологиях, что делает её эталонным примером (Reference Architecture) для создания сложных распределённых систем.

## 2. Реализованный технологический стек ("As-Built")

#### **Бэкенд:**
- **Фреймворк:** Python 3.12 + FastAPI (17 микросервисов)
- **Базы данных:**
  - **Реляционная:** PostgreSQL (для основных данных)
  - **In-Memory / Брокер:** Redis (для кэша и задач)
  - **Векторная:** Weaviate (для семантического поиска)
- **Файловое хранилище:** AWS S3 (эмулируется через LocalStack)
- **Асинхронные задачи:** Celery (с воркерами для импорта и OCR)

#### **Фронтенд:**
- **Фреймворк:** Next.js + TypeScript
- **Интернационализация (i18n):** Встроенная поддержка нескольких языков.

#### **Инфраструктура и CI/CD:**
- **Контейнеризация:** Docker + Docker Compose
- **Инфраструктура как код (IaC):** Terraform (базовая настройка)
- **Непрерывная интеграция:** GitHub Actions (базовый CI-пайплайн)

#### **Безопасность (Security):**
- **Аутентификация:** OAuth2 / JWT
- **Управление секретами:** HashiCorp Vault

#### **Наблюдаемость (Observability):**
- **Метрики:** Prometheus
- **Визуализация:** Grafana
- **Логи:** Loki + Promtail
- **Трассировка:** Jaeger + OpenTelemetry

## 3. Реализованные функциональные возможности

- **Полный цикл аутентификации:** Регистрация, вход, деактивация, администрирование.
- **Сквозной банковский сценарий:** Подключение банка -> Асинхронный импорт транзакций -> Автоматическая ML-категоризация.
- **Полный цикл работы с документами:** Загрузка в S3 -> Асинхронный OCR -> Индексация в векторной БД -> Семантический поиск на естественном языке.
- **Продвинутая аналитика:**
  - **Прогнозирование денежного потока** с визуализацией на графике.
  - **Генерация PDF-отчёта** о доходах для подачи на ипотеку.
- **Партнёрский Marketplace:** Централизованная страница для всех партнёрских услуг (бухгалтерия, страхование и т.д.) с логикой "передачи" лидов.
- **Проактивный ассистент:** Автоматическое создание напоминаний в календаре о налоговых сроках.
- **Пользовательский аудит:** Страница "Журнал активности", где пользователь видит все ключевые действия со своим аккаунтом.

## 4. Реализованные системы промышленного уровня

- **Тестирование:**
  - **Юнит-тесты** для отдельных сервисов.
  - **Контрактные тесты (Pact)** для гарантии совместимости API.
  - **Интеграционные тесты** для проверки сквозных синхронных и асинхронных сценариев.
- **Надёжность:** Асинхронная обработка задач гарантирует, что ни одна операция не будет потеряна.
- **Безопасность:** Реализованы лучшие практики по хранению паролей (хэширование), токенов (JWT) и секретов (Vault).
- **Наблюдаемость:** Развёрнут полный стек "трёх столпов" Observability, дающий беспрецедентный контроль над системой.
